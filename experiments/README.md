# Experiments Directory

This directory contains all the experiments available in this repository.

## How to Use This Directory

1. Browse the subdirectories to find experiments that interest you
2. Each experiment has its own README.md with detailed instructions
3. Follow the experiment's procedure to learn and practice
4. Modify and experiment with the code to deepen your understanding

## Available Experiments

### Getting Started
- **[example_experiment](example_experiment/)** - A simple "Hello World" program to get you started

### Templates
- **[templates](templates/)** - Templates for creating your own experiments

## Creating Your Own Experiment

Want to add a new experiment? Here's how:

1. Create a new directory with a descriptive name
2. Copy the template from `templates/experiment_template.md` to your directory as `README.md`
3. Fill in all the sections in the README
4. Add your code files and other resources
5. Test your experiment thoroughly
6. Submit a pull request (see [CONTRIBUTING.md](../docs/CONTRIBUTING.md))

## Experiment Categories

Consider organizing your experiments by category. See [EXPERIMENT_CATEGORIES.md](../docs/EXPERIMENT_CATEGORIES.md) for suggested categories.

## Tips

- **Start simple**: Begin with the example experiment
- **Read carefully**: Follow each step in the experiment's README
- **Experiment**: Don't be afraid to modify the code and see what happens
- **Learn by doing**: The best way to learn is through hands-on practice
- **Ask questions**: If something is unclear, open an issue

## Need Help?

If you have questions or run into problems:
1. Check the experiment's README for troubleshooting tips
2. Review the [CONTRIBUTING.md](../docs/CONTRIBUTING.md) guide
3. Open an issue in the repository
4. Ask for help from the community
